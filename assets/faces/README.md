# Optional family face images

These files are optional.  
If missing, the game automatically uses built-in cartoon/emoji portrait placeholders.

If you want to override placeholders, use these exact file names:

- `adult-man.png` (tank/bruiser)
- `older-woman-toddler.png` (support/healer)
- `angry-selfie-kid.png` (berserker/charger)
- `smiling-boy-car.png` (fast runner/utility)
- `staircase-family-selfie.png` (commander/title/banner)
- `pink-haired-woman-phone.png` (ranged/magic)
- `gray-haired-older-man.png` (legendary late-game)

Tips:
- Square images (at least 256x256) look best for unit faces.
- Keep style playful and non-gory.
- You can swap images anytime and refresh the page.
